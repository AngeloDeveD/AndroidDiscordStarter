# 📱 Android Discord Bot Launcher (с поддержкой Lua-модулей) 🚀

Легковесное и мощное мобильное приложение на **Kotlin**, разработанное с использованием **Jetpack Compose** и **Material Design 3 (M3)**. Позволяет в один клик запускать полностью автономного Discord-бота прямо на вашем Android-устройстве, поддерживая динамические слэш-команды (`/`), написанные на языке **Lua**.

Проект спроектирован с упором на чистую архитектуру, высокую плотность интерфейса (High Density UI) и простоту администрирования.

---

## ✨ Особенности приложения

1. **Прямое соединение (Discord Gateway v10)**  
   Бот не зависит от внешних веб-серверов или прокси-сервисов. Он соединяется напрямую с серверами Discord по защищенному протоколу веб-сокетов (**WSS**). Встроенные фоновые пинг-таймеры (Heartbeats) стабильно удерживают онлайн.
2. **JVM Lua-интерпретатор (Luaj)**  
   Динамическая компиляция и выполнение сэндбокс-скриптов Lua прямо на телефоне. Слэш-команды автоматически регистрируются в Discord API за считанные секунды.
3. **Красивый технический логгер**  
   Интерактивная консоль/терминал, разделенная по цветовым категориям логов (`INFO`, `WARNING`, `ERROR`, `SUCCESS`, `BOT`), позволяющая отслеживать веб-сокет сессии, входящие взаимодействия и ошибки рантайма Lua.
4. **Конфиденциальность и автономность**  
   Токен аутентификации сохраняется локально на устройстве в защищенных настройках `SharedPreferences`.

Благодаря декомпозиции, исходный файл `MainActivity.kt` и весь код избавлены от жестко закодированных констант или нагромождения стилей. Стили разведены по соответствующим логическим файлам:

## 🛠 Инструкция по запуску в Android Studio

Вы можете без лишней настройки импортировать и запустить данный проект. Вся усложненная конфигурация API-ключей была убрана для беспрепятственной сборки "из коробки".

### Шаг 1: Импорт проекта
1. Скачайте/клонируйте исходный код проекта.
2. Запустите **Android Studio** (рекомендуются последние версии: Koala, Ladybug или новее).
3. Нажмите **File -> Open...** (или **Import Project**) и выберите корневую папку, содержащую файл `build.gradle.kts`.
4. Подождите пару минут, пока Gradle выполнит индексацию проекта и загрузит необходимые библиотеки (`luaj-jse`, `ktor-websockets`, `compose-material3` и др.).

### Шаг 2: Запуск на устройстве / Эмуляторе
1. Включите **USB отладку (Developer Options)** на вашем физическом Android-устройстве или подготовьте виртуальное устройство AVD.
2. В верхней панели Android Studio выберите ваше устройство в выпадающем списке.
3. Нажмите зеленую кнопку **Run 'app'** (или воспользуйтесь горячими клавишами `Shift + F10`).
4. Приложение скомпилируется и успешно установится на ваш девайс.

## 🤖 Как настроить и запустить своего бота

Для работы бота вам понадобится аккаунт разработчика в Discord:

1. Откройте [Discord Developer Portal](https://discord.com/developers/applications) и авторизуйтесь.
2. Нажмите кнопку **New Application** в правом верхнем углу, введите имя и создайте проект.
3. Перейдите в раздел **Bot** (в меню слева):
   - Прокрутите страницу до **Privileged Gateway Intents** и активируйте галочки:
     - **Presence Intent**
     - **Server Members Intent**
     - **Message Content Intent**
   - Нажмите кнопку **Reset Token** и скопируйте полученную длинную строку токена.
4. Перейдите во вкладку **Installation** или **OAuth2 -> URL Generator**:
   - Во вкладке scopes установите чекбокс `bot` и `applications.commands`.
   - Во вкладке Bot Permissions выдайте необходимые права (например, `Administrator` или права на чтение/отправку сообщений).
   - Скопируйте сгенерированную ссылку внизу и откройте её в браузере, чтобы пригласить бота на ваш тестовый Discord-сервер.
5. Запустите установленное Android-приложение, вставьте в поле **"Авторизация"** скопированный токен и нажмите **"Запустить"**! Бот загорится зеленым в Discord и будет готов отвечать на слэш-команды!

## 📝 Написание и управление Lua-командами

Каждый Lua-файл в папке приложения регистрирует отдельную `/команду` в вашем Discord. Вы можете создавать их прямо во вкладке **Команды Lua** встроенного текстового редактора приложения.

### 📂 Где хранятся Lua-файлы на устройстве?
По умолчанию файлы Ваших скриптов сохраняются во внешнем хранилище приложения (External Files Directory) в папке `commands`:
* **Основной путь на диске:** `Android/data/com.aistudio.discordbot.launcher.pkg/files/commands/` (или соответствующий `applicationId` вашего билда).
* **Для просмотра и прямого переноса:** Вы можете подключить устройство к компьютеру через USB и скопировать готовые `.lua` файлы напрямую в эту папку. Также можно использовать любые современные Android-проводники (например, Solid Explorer, X-plore, CX Проводник или встроенный в Android Files).

> 🔄 **Свежая синхронизация (Hot Sync):** При добавлении или изменении Lua-файлов через компьютер или сторонний проводник вам достаточно нажать кнопку **"Синхронизировать команды"** во вкладке "Команды" либо перезапустить бота. Программа выполнит Bulk Overwrite шлюза Discord напрямую с вашего телефона, обновив команды мгновенно!

---

### Простой шаблон (`ping.lua`):
```lua
command = {
    name = "ping",
    description = "Проверить статус работы бота",
    options = {}
}

function execute(interaction)
    return "🏓 Понг! Бот успешно запущен прямо с мобильного телефона!"
end
```

### Команда с аргументом (`say.lua`):
```lua
command = {
    name = "say",
    description = "Повторить ваш текст скрытно",
    options = {
        {
            name = "text",
            description = "Введите сообщение",
            type = 3, -- STRING тип данных согласно спецификации Discord API
            required = true
        }
    }
}

function execute(interaction)
    local text = interaction.options["text"]
    return {
        content = "Вы ввели скрытый текст: ||" .. text .. "||",
        ephemeral = true -- Сообщение увидит только вызвавший команду пользователь
    }
end
```
